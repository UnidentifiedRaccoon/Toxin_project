Это простой слайдер, каждый слайд имеет:
 {
    position: absolute;
    width:100%;
    height:100%;
    background: url(...)
    animation: ...
 }

     Для корректной работы необходимо вставлять слайды без всякой обертки,
 непосредственно в блок, где требуется отобразить слайды

     В зависимости от страницы JS код создает необходимые слайды
 Другими словами вся конфигурация происходит непосредственно через JS

    Это не универсальный слайдер, а слайдер с минимальным, функционалом,
 заточенный под решение задачи конкретно этого проекта.

    Добавления слайдера на страницу в рамках этого проекта:
   ................................................................
    let page  = document.querySelector('.page');

    function getPageSlides() {
     let result = [];
     for(let i=...; i<=...; i++) {
         let div = document.createElement('div');
         div.classList.add(`slider`);
         div.classList.add(`slider__slide-${i}`);
         result.push(div);
     }
     return result;
    }

    page.prepend(...getPageSlides());
    ................................................................

    Далее стилизуйте каждый слайд с помощью CSS (не забудьте, для разного количества слайдов - разная анимация)
